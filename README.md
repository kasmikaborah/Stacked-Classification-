# Stacked-Classification
Introduced stacked classification projects for classification of cancer samples using omics datasets and performed AI-SHAP analaysis (Explainable AI).

This work is created for downstream analysis of RNA-seq and WES data

**Uniqueness of the project**
-Introduced RankXLAN, a novel explainable ensemble-basedmulti-omics framework that combines feature selection, ensemble learning, explainable AI, bioinformatics, and In-silico validation in a single pipeline. 
-Performed ML-based and statistical FS methods to identify optimal features of SC using transcriptomic and epigenomic data. 
-Identification of an effective FS method to improve the performanceof the classification model for transcriptomic and epigenomic data. 
-Ensemble learning classifier to classify SC and normal samples toevaluate the best FS techniques and identify accurate markers.
-Model validation was performed using AI-SHAP analysis. Performed bioinformatics interpretation and In-silico studytoidentify potential marker genes and targets of --SC fromtranscriptomicand epigenomic data.
